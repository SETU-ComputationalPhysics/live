Forest Fire

Aim

In this assignment we are going to implement a Cellular Automata model to study the spread of a forest fire. The aim is to reproduce the work of

A Cellular Automata Model for Fire Spreading Prediction (Quartieri:2010)1

However, we will make slight changes to their model and will put more effort into examining the effects of the various model parameters.

Outline of Assignment

  • Implement the CA model
  • Understand the model — parameter dependence
  • Apply model to real data

References

While you only need to read the Quartieri:2010 paper for this assignment I encourage you to scan the other references listed here.
