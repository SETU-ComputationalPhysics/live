Model Development and Implementation

Theory / Model

On reading the discussion in §3 of Quartieri:2010 we want the following CA model

Space

  • 2D, square grid
  • Finite with boundaries \(\implies\) will force dead zone along edge to simplify updates.

States

Slight reordering and relabeling of states from those given in paper. (TBH I have no good reason for this ☺️)

  • unburnable (UNBURNABLE=0, white)
  • burnable (BURNABLE=1, green)
  • burning (BURNING=2, red)
  • burnt (BURNT=3, black)

Neighbourhood

The paper uses Moore neighbourhood which treats all 8 neighbours with equal weight. This is reasonable, but one could argue that the diagonal neighbour cells (DN) are further away than (and so have less of an impact than) the adjacent cells (AN). However switching to a von Neumann neighbourhood is too extreme. So instead we will opt for a neighbourhood that is between Moore and von Neumann, by weighting input of diagonal neighbour cells by \(w=1/\sqrt{2}\).

Our selection of neighbourhood has a consequence on the calculation of the probability of switching from state BURNABLE to BURNING. The paper used $$ \Pr(\text{BURNABLE} \to \text{BURNING}) = \frac{N_b}{8} $$ where \(N_b\) is the number of neighbours in state BURNING. We need to modify this formula along the lines of \[ \Pr(\text{BURNABLE} \to \text{BURNING}) = \frac{AN_b+w \times DN_{b}}{4(1 + w)} \]

where \(AN_b\) is the number of adjacent neighbours in state BURNING, and \(DN_b\) is the number of diagonal neighbours in state BURNING.

We will investigate the impact of the value of our new parameter \(w\):

  • \(w=1\) will give Moore
  • \(w=0\) will give von Neumann

Note:
Feel free to simplify your implementation so that it only uses \(w=1\).

Rules

UNBURNABLE is a terminal state (never changes).

BURNT is a terminal state.

BURNABLE to BURNING

In addition to the above modification due to the neighbourhood change we will also include a probability term (or ignition rate), which control how likely a cell will switch to BURNING when neighbours are BURNING. So we have

$$ \Pr(\text{BURNABLE} \to \text{BURNING}) = p_{\text{ignition_rate}} \left(\frac{AN_b+w\cdot DN_{b}}{4(1+w)}\right) $$

So \(p_{\text{ignition_rate}}=1\) corresponds to model in paper. Smaller values should slow fire spread and could be used to represent forest conditions such as wet fuel after rain etc.

BURNING to BURNT

The paper switches from state BURNING to BURNT after one time step (next stage). This is a bit simplistic and I would prefer a model that supported staying in BURNING for a number of time steps. We have two options here:

  • we could include a counter in each cell to keep track of number of time steps it is in BURNING state. Extra data and extra code \(\implies\) don't like this.
  • we could treat the switching from state BURNING to BURNT as a geometric random variable with probability of success \(p_{\text{burnout_rate}}\)

We (you) are going to implement the geometric random variable approach and so we have

$$ \Pr(\text{BURNING} \to \text{BURNT}) = p_{\text{burnout_rate}} $$

To match paper we set \(p_{\text{burnout_rate}}=1\).

Note:
AFAIK there is an error in the paper in section 3. Bonus marks for first student to identify it.
